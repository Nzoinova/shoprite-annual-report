# NORS Shoprite Annual Service Report 2025

Interactive annual service report for the Shoprite fleet managed by NORS Trucks and Buses Angola VT.

## Live Demo
[View Report](https://YOUR_USERNAME.github.io/shoprite-annual-report/)

## Features
- Bilingual (EN/PT) toggle
- Interactive charts (Recharts)
- Animated KPI counters
- Vehicle history search & expand
- Year-over-Year comparison
- NORS brand system (monochromatic + Trucks & Buses accents)
- Responsive & print-friendly

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- Recharts 3
- Framer Motion
- Lucide Icons

## Deployment
Push to `main` branch triggers automatic deployment to GitHub Pages via GitHub Actions.

### Manual Setup
```bash
npm install
npm run dev      # Development server on :3000
npm run build    # Production build to /dist
```

## Data Period
January 2, 2025 - February 26, 2026 (14 months, 345 interventions, 57 vehicles)
